Винительный отвечает на вопросы «кого? что?» (Koga? Šta?). Это падеж прямого дополнения и направления движения.

Когда используется:

- **Прямой объект** (вижу кого/что, люблю кого/что): `Видим кућу`, `Волим маму`.
- **Направление** (куда?) с предлогами `у`, `на`: `Идем у школу`, `Стављам на сто`.
- **Время** (в какой день, на какое время): `У понедељак`, `За викенд`.
- **Сквозь/Через/За**: `Кроз шуму` (через лес), `За мене` (для меня).

Частые предлоги:

- `у`, `на` — куда (динамика): `у град`, `на море`.
- `кроз` — сквозь/через: `кроз прозор`.
- `уз` — вверх по/вдоль: `уз брдо`.
- `низ` — вниз по: `низ улицу`.
- `за` — для / за (кого-то) / через (время): `Ово је за тебе`.

Окончания (упрощённо):

- **m.sg**:
    - Одушевлённые (= Genitiv): -а → `видим брата`.
    - Неодушевлённые (= Nominativ): -ø → `видим град`.
- **f.sg** на -а: -у → `жена` → `жену`, `књига` → `књигу`.
- **n.sg**: (= Nominativ): -о/-е → `село`, `море`.
- **pl**:
    - m: -е → `градове`.
    - f: -е → `жене`.
    - n: -а → `села`.

Типичные конструкции:

- `Ићи у/на + Acc.` — Идти в/на...
- `Волети/Гледати/Слушати + Acc.` — Любить/Смотреть/Слушать кого/что.
- `За + Acc.` — Для кого-то.

Примеры (ћирилица / latinica → перевод):

- **Видим твог брата.** / Vidim tvog brata. → Вижу твоего брата.
- **Гледам филм.** / Gledam film. → Смотрю фильм.
- **Идем у Београд.** / Idem u Beograd. → Еду в Белград.
- **Купујем воду.** / Kupujem vodu. → Покупаю воду.
- **Хвала за помоћ.** / Hvala za pomoć. → Спасибо за помощь.

Подсказки

- У мужского рода важно различать одушевлённое (как Genitiv) и неодушевлённое (как Nominativ).
- Женский род на `-а` всегда меняется на `-у`.
