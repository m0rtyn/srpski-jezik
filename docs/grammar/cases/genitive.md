Родительный отвечает на вопросы «кого? чего?» и часто передаёт принадлежность, часть от целого, исходную точку движения и объект при отрицании.

Когда используется:

- Принадлежность и связь двух существительных: `книга професора`, `кључеви аута`.
- После количественных слов (много, мало, бројки > 4) и мер: `чаша чаја`, `пет књига`.
- Объект отрицательного глагола: `не видим брата`, `нисам чула звук`.
- После некоторых глаголов/прилагательных, требующих части/отсутствия: `бојим се висине`, `пун је енергије`.
- С обязательными предлогами: `из`, `од`, `без`, `код`, `до`, `уместо`, `поред`, `изнад`, `испод`, `иза`, `испред` (в статичном смысле) и др.

Частые предлоги + значения:

| Предлог | Значение |
| --- | --- |
| `из` | откуда: `из града` |
| `од` | от, от кого: `писмо од Милоша` |
| `без` | без кого/чего: `без шећера` |
| `код` | у кого / возле: `код пријатеља` |
| `до` | до точки: `до школе` |
| `уместо` | вместо: `уместо кафе` |
| `поред` | рядом с: `поред куће` |
| `иза/испред` | позади / впереди (статично) |

Окончания (упрощённо):

- m.sg: -а → `град` → `града`, `брат` → `брата`.
- f.sg на -а: -е → `жена` → `жене`; на -ост/-т: -и → `радост` → `радости`.
- n.sg: -а → `село` → `села`, `море` → `мора`.
- pl: чаще всего -а (m/n) и -а/-и (f): `градова`, `села`, `жена`. Для слов на -иц/-ост возможны варианты `-и`: `улица` → `улица`, `радост` → `радости`.

Типичные конструкции:

- Количество + Gen.: „три студента“, „пуно времена“.
- `бојати се + Gen.`: „бојим се мрака“.
- Отрицание + Gen.: „не познајем тог човека“.
- `из/од/без/код/до + Gen.`

Примеры (ћирилица / latinica → перевод):

- Немам времена. / Nemam vremena. → У меня нет времени.
- Вратила сам се из града. / Vratila sam se iz grada. → Я вернулась из города.
- Бојим се висине. / Bojim se visine. → Я боюсь высоты.
- Чекамо код пријатеља. / Čekamo kod prijatelja. → Ждём у друга.
- Пијем шољу чаја. / Pijem šolju čaja. → Пью чашку чая.
- Без шећера, молим. / Bez šećera, molim. → Без сахара, пожалуйста.

Подсказки

- После числительных 2–4 часто используется форма на `-а/-е`, но с прилагательным возможен синтаксис „две лепе књиге“ (Nom.) + „књиге“ (Gen.).
- Если нет существительного для количества, используйте собирательное: „петоро деце“ (Gen. „деце“ неизменяемо).
- В отрицательных предложениях Gen. не обязателен, но предпочтителен в формальном стиле: „Не видим брата“ vs „Не видим брата/брата“ (вариант Acc.).
