Предложный отвечает на вопросы «где? о ком? о чём?» (Gde? O kome? O čemu?). Используется **только с предлогами**.

Когда используется:

- **Местонахождение** (где? — статика): `Живим у граду`, `Књига је на столу`.
- **Объект мысли/речи** (о ком/чём?): `Причамо о филму`.
- **Предлоги**: `у`, `на`, `о`, `по`, `при`.

Частые предлоги:

- `у` — в (внутри): `у школи`.
- `на` — на (поверхности): `на столу`.
- `о` — о (об): `о љубави`.
- `по` — по (распределение): `по граду`.
- `при` — при (возле): `при крају` (в конце).

Окончания (упрощённо):

- **m.sg**: -у → `град` → `у граду`, `аеродром` → `на аеродрому`.
- **f.sg** на -а: -и → `школа` → `у школи`, `соба` → `у соби`.
- **n.sg**: -у → `село` → `у селу`.
- **pl**: -има/-ама (совпадает с Dativ и Instrumental): `градовима`, `школама`, `селима`.

Типичные конструкции:

- `Живети у...` — Жить в...
- `Бити на...` — Быть на...
- `Говорити о...` — Говорить о...

Примеры (ћирилица / latinica → перевод):

- **Живим у Београду.** / Živim u Beogradu. → Я живу в Белграде.
- **Кључ је на столу.** / Ključ je na stolu. → Ключ на столе.
- **Размишљам о теби.** / Razmišljam o tebi. → Думаю о тебе.
- **Шетамо по парку.** / Šetamo po parku. → Гуляем по парку.

Подсказки

- Предлоги `у` и `на` используются и в Винительном (куда?), и в Предложном (где?). Различайте по глаголу:
    - `Идем у школу` (Acc., Куда? Динамика).
    - `Седим у школи` (Loc., Где? Статика).
- Окончания единственного числа мужского и среднего рода (`-у`) совпадают с Дательным.
