Краткий обзор всех 7 падежей сербского языка.

| Падеж | Вопросы (SR) | Вопросы (RU) | Основная функция | Окончания (m.sg / f.sg / n.sg) |
| --- | --- | --- | --- | --- |
| **1. Nominativ** | Ko? Šta? | Кто? Что? | Подлежащее, словарь | Ø / -a / -o, -e |
| **2. Genitiv** | Koga? Čega? | Кого? Чего? | Принадлежность, «нет...», «из...» | -a / -e / -a |
| **3. Dativ** | Kome? Čemu? | Кому? Чему? | Адресат, «к...» | -u / -i / -u |
| **4. Akuzativ** | Koga? Šta? | Кого? Что? | Прямой объект, «в/на» (куда) | -a (одуш), Ø (неод) / -u / -o, -e |
| **5. Vokativ** | — | — | Обращение | -e, -u / -o, -a / -o, -e |
| **6. Instrumental** | S kim? Čime? | Кем? Чем? | Инструмент, «с...» | -om / -om / -om |
| **7. Lokativ** | Gde? O kome? | Где? О ком? | Место (где), «в/на» (где) | -u / -i / -u |

### Заметки по окончаниям

1.  **Dativ и Lokativ** в единственном числе часто совпадают (`-u` для m/n, `-i` для f). Разница в предлогах (Lokativ всегда с предлогом).
2.  **Dativ, Instrumental, Lokativ** во множественном числе имеют одинаковые окончания:
    - m/n: `-ima` (`gradovima`, `selima`)
    - f: `-ama` (`ženama`)
3.  **Akuzativ** мужского рода ед. числа:
    - Одушевлённое = Genitiv (`-a`).
    - Неодушевлённое = Nominativ (`Ø`).

### Порядок изучения

Обычно падежи учат в таком порядке (по частоте и логике):
1.  Nominativ (база)
2.  Akuzativ (самый частый: «вижу», «иду в»)
3.  Lokativ (где я?)
4.  Genitiv (откуда? чей? сколько?)
5.  Instrumental (с кем? чем?)
6.  Dativ (кому? мне нравится)
7.  Vokativ (обращение)

См. подробнее в разделах по каждому падежу.
